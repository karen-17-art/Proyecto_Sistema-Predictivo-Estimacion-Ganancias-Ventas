# SISTEMA PREDICTIVO DE ESTIMACIÓN DE GANANCIAS DE VENTAS

---

#  INTRODUCCIÓN

En los últimos años, el comercio electrónico ha presentado un crecimiento constante a nivel mundial, especialmente en el sector moda, donde las empresas utilizan descuentos y promociones como estrategias para aumentar las ventas.

Sin embargo, muchas decisiones comerciales todavía se realizan de manera empírica, sin herramientas analíticas que permitan estimar el impacto económico de dichas estrategias. Esto puede generar pérdidas económicas y afectar la rentabilidad de las empresas.

Frente a esta problemática, el presente proyecto propone el desarrollo de un sistema predictivo basado en Ciencia de Datos y Machine Learning para estimar las ganancias generadas por las ventas dentro de un entorno ecommerce.

---

#  ¿DE QUÉ TRATA EL PROYECTO?

El proyecto consiste en desarrollar un sistema predictivo capaz de analizar información comercial y estimar las ganancias obtenidas por cada venta en ecommerce de moda.

Para ello, se utilizan técnicas de análisis de datos, visualización estadística y modelos predictivos que permiten identificar relaciones entre descuentos, productos, campañas promocionales y ganancias.

Además, el proyecto incorpora una mini aplicación web interactiva desarrollada en Python, permitiendo visualizar información y generar predicciones de manera dinámica.

---

#  EXPLICACIÓN DE LA DATA

La información utilizada proviene de un dataset público obtenido desde Kaggle relacionado con ventas ecommerce.

La data se encuentra distribuida en diferentes archivos CSV que contienen información sobre:

* Ventas
* Productos
* Clientes
* Campañas promocionales
* Stock
* Canales de venta

Durante el desarrollo del proyecto se realizó un proceso de:

* Limpieza de datos
* Integración de tablas
* Transformación de variables
* Tratamiento de valores atípicos
* Análisis exploratorio de datos (EDA)

A partir de ello, se construyó un dataset analítico final con variables como:

* Ganancia total
* Precio promedio
* Nivel de descuento
* Categoría principal
* Canal de venta
* Campañas activas
* Mes de venta
* Día de la semana
* Cantidad de productos vendidos

---

#  FINALIDAD DEL PROYECTO

La finalidad principal del proyecto es ayudar a mejorar la toma de decisiones comerciales dentro de empresas ecommerce del sector moda.

El sistema busca anticipar cómo influyen los descuentos y promociones en las ganancias obtenidas, permitiendo reducir pérdidas económicas y optimizar estrategias comerciales.

Asimismo, se busca fomentar el uso de herramientas de análisis de datos y Machine Learning para apoyar decisiones empresariales basadas en información real.

---

#  OBJETIVO GENERAL

Desarrollar e implementar un modelo predictivo basado en técnicas de Machine Learning y analítica de datos que permita estimar la ganancia de ventas en ecommerce de moda.

---

#  OBJETIVOS ESPECÍFICOS

* Analizar y preparar los datos comerciales mediante procesos de limpieza y transformación.

* Identificar variables relacionadas con las ganancias por venta.

* Desarrollar modelos predictivos utilizando técnicas de regresión.

* Evaluar el rendimiento del modelo mediante métricas estadísticas.

* Implementar una mini aplicación web interactiva para visualizar predicciones.
